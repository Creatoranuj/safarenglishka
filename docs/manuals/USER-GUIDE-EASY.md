# 📱 Safar Englishka — Student Guide (Aasan Bhasha)

Ye guide bilkul simple Hinglish me hai. Har section me "kya karna hai" aur "kya milega" likha hai. Technical baate nahi — bas step-by-step.

---

## 1. App kaise install kare

**3 tarike hain:**

### A. Play Store se (sabse aasan)
1. Phone me Play Store kholein.
2. Search karein: **"Safar Englishka"**.
3. Install button dabayein.
4. Ho gaya — home screen par icon dikhega.

### B. Direct APK se
1. Browser me kholein: [github.com/MrAnujBabu/Safar-Englishka-Install](https://github.com/MrAnujBabu/Safar-Englishka-Install/releases)
2. Latest version me **APK Download** par tap karein.
3. Phone bolega "Install unknown apps" — ek baar allow kar do.
4. Install ho jaayega.

### C. Bina install kiye — Browser me hi chalao (PWA)
1. Chrome/Firefox me website kholein.
2. Menu (3 dot) → **"Add to Home Screen"** dabayein.
3. Ab home par icon aa jaayega — bina Play Store ke.

## 2. Account kaise banaye

1. App kholte hi **Sign Up** par tap karein.
2. Email ya Mobile number daale.
3. OTP aayega — 6 digit ka code daalein.
4. Password set karein (kam se kam 8 letters).
5. Naam, Class, Board daalein.
6. ✅ Ho gaya — ab aap login hain.

**Bhool gaye password?** Login page par **"Forgot Password"** — email par reset link aayega.

---

## 3. Course kaise dhundhe aur khareede

### Free course
1. Home → **Courses** tab.
2. Jo course pasand hai uspe tap karein.
3. **Enroll Free** button dabayein.
4. Turant khul jaayega — Lecture dekh sakte hain.

### Paid course
1. Course kholein.
2. Neeche **Buy Now** — price dikhega.
3. Payment page khulega (Razorpay).

---

## 4. Payment kaise kare

**UPI (recommended)** — GPay, PhonePe, Paytm, BHIM.
**Card** — Debit/Credit.
**Net Banking** — sabhi bade banks.

### Payment steps
1. Method choose karein.
2. UPI PIN / OTP daalein.
3. **Success** message dikhne tak wait karein — page apne aap wapas aayega.
4. Course "My Courses" me aa jaayega.

### Agar payment fail ho?
- Paisa kata? — 5-7 din me apne aap refund ho jaayega.
- "Bank could not verify" — bank ka 3D-Secure fail, dobara try karein.
- 24 ghante intezaar karein; agar phir bhi access na mile → Support par WhatsApp bhejein order-ID ke saath.

---

## 5. Lecture kaise dekhe (Video Player)

### Basic controls
- **Play / Pause** — video ke bich me tap
- **10s aage** — right side me double-tap
- **10s peeche** — left side me double-tap
- **Speed change** — settings icon → 0.5x / 1x / 1.5x / 2x
- **Fullscreen** — bottom-right ka square icon
- **Volume / brightness** — right/left side me upar-neeche swipe
- **Fast forward hold** — screen ko long press → 2x speed

### Fullscreen me
- Ek tap → controls dikhenge + system nav bar aayega
- Dobara tap → dono hide
- Back button → pehle rotation reset, phir exit

`[Screenshot: video player controls]`

### Watermark
Aapka naam + phone number video par light dikhata hai — piracy rokne ke liye. Ye normal hai.

## 6. Notes, PDF, DPP kaise download kare

1. Lecture page me neeche **Notes** / **DPP** / **PDF** tab.
2. Har item ke aage **download icon** ⬇️.
3. Tap karein — offline access ke liye save ho jaayega.
4. Downloaded PDF **"My Library"** me milega — internet na ho tab bhi khulega.

**Storage full?** Old PDFs delete karein: My Library → long press → Delete.

---

## 7. Quiz kaise de aur result kaise dekhe

### Quiz attempt
1. Course → Quiz tab.
2. **Start Quiz** dabayein.
3. Har question par option choose karein → **Next**.
4. Time limit hai to timer upar dikhega.
5. Last me **Submit** dabayein.

### Result review
- Score turant dikhega (X/Total).
- **Review Answers** dabayein — har question ka sahi jawab aur explanation dikhega.
- Galat jawab **laal**, sahi **hara**.

### Retake
Kuch quizzes me multiple attempts allowed hain — dobara **Start Quiz** dabayein.

---

## 8. Doubt kaise pooche

### A. AI Sarthi (turant reply)
1. Bottom right me chat icon 💬.
2. Apna sawal Hindi ya English me likhein.
3. AI turant answer dega + related videos suggest karega.

### B. Community / Teacher
1. Lecture page → **Ask Doubt**.
2. Photo attach kar sakte hain (question ka).
3. Teacher reply karega — notification aayega.

### C. Community forum
Home → **Community** — dusre students se discuss karein.

---

## 9. Live class kaise join kare

1. Home → **Live** tab.
2. Aane wali classes ki list dikhegi with **time**.
3. **Set Reminder** — 10 min pehle notification aayega.
4. Time par **Join Live** button active hoga → tap karein.
5. Chat me sawal poochh sakte hain live.

---

## 10. Profile, Password, Logout

- **Profile** — top right avatar → naam, photo, class edit karein.
- **Password change** — Settings → Security → Change Password.
- **Logout** — Settings ke sabse neeche.
- **Account delete** — Settings → Privacy → Delete Account (30 din me hoga).

---

## 11. Common problems aur solutions

| Problem | Solution |
| ------- | -------- |
| Video nahi chal raha | Internet check → app restart → phir bhi nahi to WhatsApp |
| PDF nahi khul raha | Storage space check → dobara download |
| Payment kata, access nahi mila | 5 min wait → "My Courses" refresh → 24hr me apne aap aa jaayega |
| App crash ho raha | App update check → phone restart → cache clear (Settings → Storage) |
| Login nahi ho raha | "Forgot Password" → OTP dobara maangein |
| Notification nahi aa raha | Phone Settings → Apps → Safar Englishka → Notifications ON |
| Screen record block hai | Ye jaan-boojh ke hai — content protection ke liye |
